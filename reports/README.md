# Reports

Human-readable STT comparison reports generated from PUnit explore results:

```
reports/stt-comparison.html
reports/stt-comparison.md
```

Generated by:

```bash
./gradlew generateSttReport
```

The reporting layer (`org.mavai.sttbench.report.SttReportGenerator`)
consolidates multiple explore result files into a single provider × recipe
comparison. It is an STT-specific consumer of PUnit output, **not** a generic
PUnit reporting subsystem — for full PUnit reporting use `punit-report`.

Reports are **gitignored** — they are reproducible from the results. Only this
README is tracked.
